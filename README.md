# Sprint 7 - Dashboard de Vehículos Usados
## Descripción

Esta aplicación web fue desarrollada con Streamlit y Plotly como parte del Sprint 7 del  Bootcamp de Ciencia de Datos de Tripleteen.

Permite explorar un conjunto de datos de anuncios de venta de vehiculos usados mediante visualizaciones interactivas.

## Tecnologías 

-Python
-Pandas
-Plotly
-Streamlit

## Funcionalidades

-Visualización del conjunto de datos
-Histograma del Odómetro
-Grafico de Dispersión Precio vs Odómetro 
-Filtro por tipo de vehículo

## Cómo ejecutar

```bash
streamlit run app.py
```
